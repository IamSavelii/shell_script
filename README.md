# shell_script

This script can copy, move, delete and rename files.

  

### Installation


```sh
$ git clone https://github.com/IamSavelii/shell_script.git
$ cd shell_script
```

### Lauch

```sh
$ sh script.sh
```

### Help

| Key | Description |
| ------ | ------ |
| -c | Copy file |
| -r | Rename file |
| -m | Move file |
| -d | Delete file |
| --help | Help |
